## ðŸ“š Basic Formatting

### **Bold Text**
Make text stand out by wrapping it in double asterisks:
```
**This text will be bold**
```
Result: **This text will be bold**

### *Italic Text*
Make text italic by wrapping it in single asterisks:
```
*This text will be italic*
```
Result: *This text will be italic*

### ***Bold AND Italic***
Combine both with triple asterisks:
```
***This text is bold and italic***
```
Result: ***This text is bold and italic***

### ~~Strikethrough~~
Cross out text with double tildes:
```
~~This text is crossed out~~
```
Result: ~~This text is crossed out~~

## ðŸ”¢ Lists

### Bullet Points (Unordered Lists)
Create bullet points with hyphens, asterisks, or plus signs:

```
- First item
- Second item
- Third item
  - Nested item (add 2 spaces before the dash)
  - Another nested item
```

Result:
- First item
- Second item
- Third item
  - Nested item
  - Another nested item

### Numbered Lists (Ordered Lists)
Create numbered lists like this:

```
1. First step
2. Second step
3. Third step
   1. Sub-step (add 3 spaces before the number)
   2. Another sub-step
```

Result:
1. First step
2. Second step
3. Third step
   1. Sub-step
   2. Another sub-step

**ðŸ’¡ Tip:** You can use `1.` for all items and Markdown will auto-number them!

```markdown
Phishing is a **cybercrime** technique where scammers try to trick you into giving away personal information.

## Common Signs of Phishing

Look out for these warning signs:

1. **Urgent language** - Messages that pressure you to act quickly
2. **Suspicious links** - URLs that don't match the company's official website
3. **Poor grammar** - Legitimate companies usually have professional writing
4. **Requests for personal info** - Banks will *never* ask for passwords via email

> **Remember:** When in doubt, don't click! Contact the company directly using a phone number from their official website.

### How to Protect Yourself

- Never click links in suspicious emails
- Always verify the sender's email address
- Use two-factor authentication when possible
- Keep your software updated

---

*Stay safe online!*
```

## ðŸŽ¨ Best Practices

### âœ… DO:
- Use headings to organize your content
- Break up long paragraphs into smaller chunks
- Use bullet points for lists of items
- Use numbered lists for sequential steps
- **Bold** important terms when first introduced
- Add *emphasis* to key points
- Use quotes for important warnings or tips

### âŒ DON'T:
- Use too many headings (keeps things simple)
- Make everything bold (it loses impact)
- Create walls of text (break it up!)
- Forget to preview your content before saving
